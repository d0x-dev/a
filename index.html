How to Install

Get up and running in under a minute.

1

Download the App

Choose your platform above and click Download. Android users get a signed APK; Windows users can pick Setup or Portable.

2

Allow Installation (Android)

Go to Settings → Security → Unknown Sources and enable installation from unknown sources.

3

Install the App

Open the downloaded file from your notifications or file manager and follow the prompts.

4

Launch & Enjoy

Open AirBeats and start enjoying a premium music experience, free forever!

Frequently Asked Questions

Is AirBeats really free?
Yes. AirBeats is 100% free and open source with no ads, subscriptions, or hidden costs.
Which platforms are supported?
Android 8.0+ (signed APK) and Windows 10/11 (Setup installer or Portable version).
Is the APK safe to install?
All releases are built and signed from the public source code on GitHub. You can audit the code yourself or build it from source.
How can I contribute?
Fork the repository on GitHub, make your changes, and open a pull request. Bug reports and feature requests via GitHub Issues are also welcome.
Where do I report bugs?
Open an issue at github.com/d0x-dev/AirBeats/issues with details about your device and the problem.
